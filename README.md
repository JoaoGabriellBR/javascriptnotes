

## Sobre o projeto

https://sitejavascriptnotes.netlify.app

Javascript Notes se trata de uma aplicação CRUD (criar, ler, atualizar e excluir) full stack voltada para a
criação de notas. 

<br> 



## Tecnologias utilizadas

<ul>
    <li>ReactJs</li>
    <li>JavaScript</li>
    <li>NodeJS</li>
    <li>Express</li>
    <li>MongoDB</li>
    <li>Bulma</li>
</ul>
